Getting Started

  1. Enter your URL - Type or paste your website URL in the input field
  2. Click Analyze - Hit the "Analyze Website" button to start the compatibility check
  3. Review Results - Get detailed insights across four key areas
  4. Implement Changes - Follow our recommendations to improve compatibility

Common Issues & Solutions

Low Reader Mode Score

  • Use semantic HTML tags (article, section, header, main)
  • Implement proper heading hierarchy (H1, H2, H3...)
  • Add alt text to all images
  • Structure content with clear paragraphs

Poor Apple Intelligence Score

  • Write clear, focused content
  • Use descriptive headings and subheadings
  • Add Schema.org markup for content understanding
  • Minimize distracting elements

Missing Structured Data

  • Implement JSON-LD structured data
  • Add Article or WebPage schema
  • Include author and publication dates
  • Validate markup with Google's Structured Data Testing Tool

Best Practices

  • Test regularly - Check your site after major content or design changes
  • Focus on content - Prioritize clear, well-structured content over decoration
  • Use semantic HTML - Proper HTML structure benefits all scoring areas
  • Implement gradually - Start with high-priority recommendations
  • Validate changes - Re-test after implementing improvements

Troubleshooting

Analysis Failed

If the analysis doesn't complete:

  • Ensure the URL is publicly accessible
  • Check for CORS restrictions
  • Try again with a different page
  • Make sure JavaScript is enabled

Unexpected Scores

If scores seem incorrect:

  • Review the detailed recommendations
  • Check that the page loaded completely
  • Consider dynamic content that may not be analyzed
  • Test with a simpler page structure first
