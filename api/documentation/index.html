API Endpoints

Analyze Website

POST /analyze

Analyzes a single website for Safari Reader Mode and Apple Intelligence compatibility.

Request Body

{
  "url": "https://example.com/article",
  "options": {
    "includeRecommendations": true,
    "detailedAnalysis": true,
    "checkAccessibility": true,
    "analyzeImages": false,
    "timeout": 30
  }
}

Parameters

Parameter Type Required Description
url string Yes The URL to analyze (must be publicly accessible)
options.includeRecommendations boolean No Include optimization recommendations (default: true)
options.detailedAnalysis boolean No Include detailed analysis breakdown (default: false)
options.checkAccessibility boolean No Include WCAG accessibility analysis (default: true)

Analysis History

GET /history

Retrieves analysis history for your account with filtering and pagination options.

Query Parameters

Parameter Type Description
limit integer Number of results (default: 50, max: 200)
offset integer Pagination offset (default: 0)
url string Filter by specific URL or domain
from string Start date (ISO 8601 format)
to string End date (ISO 8601 format)

Example Request

GET /history?limit=10&url=example.com&from=2024-12-01

Error Handling

The API uses conventional HTTP response codes to indicate success or failure:

HTTP Code Error Code Description
200 - Success
400 INVALID_REQUEST Invalid request parameters
401 UNAUTHORIZED Invalid or missing API key
403 FORBIDDEN API key doesn't have required permissions
404 NOT_FOUND Resource not found
429 RATE_LIMITED Rate limit exceeded
500 INTERNAL_ERROR Internal server error

Code Examples

JavaScript (Node.js)

const fetch = require('node-fetch');

async function analyzePage(url) {
  const response = await fetch('https://api.surfacescore.com/v1/analyze', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      url: url,
      options: {
        includeRecommendations: true,
        detailedAnalysis: true
      }
    })
  });
  
  const data = await response.json();
  
  if (data.success) {
    console.log('Overall Score:', data.data.scores.overall);
    console.log('Reader Mode Score:', data.data.scores.readerMode);
    console.log('Apple Intelligence Score:', data.data.scores.appleIntelligence);
  } else {
    console.error('Analysis failed:', data.error.message);
  }
}

analyzePage('https://example.com/article');

Python

import requests
import json

def analyze_page(url):
    headers = {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    }
    
    payload = {
        'url': url,
        'options': {
            'includeRecommendations': True,
            'detailedAnalysis': True
        }
    }
    
    response = requests.post(
        'https://api.surfacescore.com/v1/analyze',
        headers=headers,
        json=payload
    )
    
    data = response.json()
    
    if data['success']:
        scores = data['data']['scores']
        print(f"Overall Score: {scores['overall']}")
        print(f"Reader Mode Score: {scores['readerMode']}")
        print(f"Apple Intelligence Score: {scores['appleIntelligence']}")
    else:
        print(f"Analysis failed: {data['error']['message']}")

analyze_page('https://example.com/article')

Webhooks

Receive real-time notifications when analysis completes or scores change significantly.

Webhook Events

  • analysis.completed - Single analysis finished
  • batch.completed - Batch analysis finished
  • score.changed - Significant score change detected
  • error.occurred - Analysis error occurred

Webhook Payload

{
  "event": "analysis.completed",
  "timestamp": "2024-12-18T10:30:00Z",
  "data": {
    "analysisId": "analysis_abc123",
    "url": "https://example.com/article",
    "scores": {
      "overall": 85,
      "readerMode": 92,
      "appleIntelligence": 78
    }
  }
}
